# Influence of Unemployment and GDP Growth Rate on SP 500 Return Rates

## Overview
This project analyzes the relationship between unemployment rate changes, GDP growth, and S&P 500 returns using time series econometric models.

## Data
- Unemployment Rate (UNRATE) – Federal Reserve Bank of St. Louis (FRED)
- Gross Domestic Product (GDP) – FRED
- S&P 500 Index – Yahoo Finance

The dataset covers the period 1960–2025
## Methods
- Unit root testing (DF-GLS)
- Vector Autoregression (VAR)
- Structural VAR with Cholesky decomposition
- Impulse Response Functions
- Granger causality testing
## Key Findings
The results suggest that unemployment rate changes and GDP growth have little influence on S&P 500 returns. In contrast, shocks to S&P 500 returns have a significant effect on future stock market performance. Overall, the findings indicate that these macroeconomic indicators do not provide strong predictive power for stock market returns and appear to move largely independently from market performance.
